# Portfólio Profissional Interativo

Este projeto é um **hub de carreira estático** desenvolvido com foco em boas práticas de HTML5, CSS3 e acessibilidade. Ele apresenta informações profissionais, projetos e um canal de contato funcional.

## 🧩 Funcionalidades

* ✅ Página única (Landing Page)
* ✅ Seção “Sobre” com foto e biografia
* ✅ Portfólio com cards de projetos
* ✅ Links externos para repositórios
* ✅ Formulário de contato funcional (Formspree)
* ✅ Ícones sociais (GitHub e LinkedIn)
* ✅ Botão “voltar ao topo”
* ✅ Navegação suave entre seções

## 🛠️ Tecnologias utilizadas

* HTML5 (semântica)
* CSS3 (Flexbox + Grid)
* SVG (ícones vetoriais)
* Formspree (envio de formulário)

## 📱 Responsividade

O layout foi desenvolvido com abordagem **mobile-first**, garantindo:

* 📱 360px (smartphones)
* 💻 1280px+ (desktops)

Sem rolagem horizontal e com boa legibilidade.

## ♿ Acessibilidade

* Uso de tags semânticas (`header`, `main`, `section`, etc.)
* Imagens com atributo `alt`
* Contraste de cores adequado
* Navegação clara e estruturada

## ⚡ Performance

O projeto foi otimizado para obter boa pontuação no Lighthouse:

* Código limpo
* CSS organizado
* Sem dependência de frameworks
